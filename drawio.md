**draw.io の XML** 作成

## 最初の動作

最初のメッセージでは、次の質問だけをしてください：
-「図の元になるメモのパスをすべて教えてください。また、ファイルの中の特定の場所からであればその内容も教えてください。」
それ以外の出力は禁止です。

---

## 手順

1. ファイルパスや具体的な指示を受け取ったら、以下を簡潔にまとめて提示してください。
   要素間の関係について、文言では伝わりにくいので、なるべくグラフィカルに表現してください。

   * 登場する要素
   * 要素間の関係
   * 出力する図の形式（あなたが解釈した形式：例：フロー図、システム構成図、ER 図など）
   * あいまいな点（あれば質問）

   要素間の関係は、テキストのみで視覚的に理解しやすいように以下の形式を活用し、グラフィカルに表現してください。

   * **ASCIIアート（箱と矢印）**
     例：

     ```
     [A] → [B] → [C]
     ```

   * **階層構造（インデント）**

     ```
     A
       ├─ A-1
       └─ A-2
     ```

   * **接続表（from/to/関係）**

     ```
     - A → B : データ送信
     - B → C : バッチ処理
     ```


2. 回答を受領し曖昧な点が解消したら、要素と関係の最終版をまとめ直し、
   -「この内容でdrawioファイルを作ってよいですか？」
   と確認してください。

3. 許可が出たら、tempフォルダ配下にファイルを出力してください。
